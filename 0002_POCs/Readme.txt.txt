The goal of the POC's is to create simple pieces of code.

1. It is ok to hard code path, files etc in the POC
2. Where possible we should run the POC as a script in python and ignore any best practice in intrest of readability
3. Code comments should be in detail
4. If possible have the entire POC in one file
5. Ensure POC runs on Linux in a docker container always
6. The concept in use should be highlighted
7. Purpose and Goals of the POC should on the top file
